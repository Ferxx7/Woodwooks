# Paleta de Colores - Wooks

## Nueva Paleta Aplicada

Esta es la paleta de colores que se ha aplicado en todo el proyecto:

### Colores Principales

1. **#546387** - Azul/Morado Oscuro (Deep Blue-Purple)
   - Uso: Textos principales, fondos oscuros, elementos de navegación
   - RGB: rgba(84, 99, 135, ...)

2. **#827E63** - Beige/Café (Warm Beige-Brown)
   - Uso: Color secundario, botones, acentos, precios
   - RGB: rgba(130, 126, 99, ...)

3. **#A4623E** - Naranja/Terracota (Terracotta Orange)
   - Uso: Acentos, hover states, llamadas a la acción
   - RGB: rgba(164, 98, 62, ...)

4. **#F2EDDE** - Beige Claro/Crema (Cream Beige)
   - Uso: Fondos claros, áreas de contenido
   - RGB: rgba(242, 237, 222, ...)

## Verificación

Para verificar que los cambios se aplicaron correctamente:

1. Abrir `index.html` en un navegador
2. Verificar que los colores coincidan con la paleta
3. Abrir `domotica.html` en un navegador
4. Verificar que los colores coincidan con la paleta
5. Revisar todos los estados hover
6. Verificar la legibilidad del texto

## Notas

- La paleta se ha aplicado de manera coherente en todo el sitio
- Los colores mantienen un buen contraste para accesibilidad
- Los gradientes usan combinaciones armoniosas de la paleta
- Los efectos hover usan variaciones de los colores principales
